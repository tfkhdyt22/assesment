This repository contains API automation tests using **Playwright** with the built-in `@playwright/test` runner.  
The test cases are based on the assessment requirements using the public API: https://reqres.in.

## Test Coverage
- GET User (Positive & Negative)
- PUT Update User (Positive & Negative)
- DELETE User (Positive & Negative)

## Command Execution

To execute the API automation tests:

1. Run all tests:
npx playwright test

2. Show the HTML report:
npx playwright show-report
